## 摘要
随着大语言模型被部署为通信智能体，智能体间的通信是否会导致输出趋同？我们提出BOUNDARY_SYNC协议，通过耦合放大因子（CAF = JSD_cond / JSD_baseline）测量表征耦合，其中CAF < 1表示同质化，CAF > 1表示多样化。在受控的GPT-4o实验（N=30，约9,900次API调用）中，我们测量了文本和图像通信中的耦合。主要发现：（1）文本通信导致显著同质化（CAF=0.803 [0.740, 0.873]，d=1.30，p<0.001），通过无通信消融和提示扰动控制得到验证；（2）图像通信在模态内基线下也呈现同质化（CAF=0.834 [0.811, 0.858]），效应比例相当；（3）群体规模调节耦合方向——K=5产生同质化，而K=3产生CAF > 1.0（点估计1.14和1.06，置信区间待定），表明向多样化的方向转变；（4）跨模型复现呈现极端变异（CAF 0.034-0.803），DeepSeek受格式伪影主导；（5）耦合是无状态的——由提示上下文而非累积更新驱动，连续共识产生单调收敛。这些结果表明，LLM智能体耦合在提示层面是真实、可测量且可控的，对多智能体系统设计具有直接意义。

## Abstract
As large language models (LLMs) are deployed as communicating agents, does inter-agent communication cause outputs to converge? We introduce BOUNDARY_SYNC, a protocol measuring representational coupling via the Coupling Amplification Factor (CAF = JSD_cond / JSD_baseline), where CAF < 1 indicates homogenization and CAF > 1 indicates diversification. In controlled GPT-4o experiments (N=30, ~9,900 API calls), we measure coupling in text and image communication. Key findings: (1) text communication causes significant homogenization (CAF=0.803 [0.740, 0.873], d=1.30, p<0.001), confirmed by no-communication ablation and prompt-perturbation controls; (2) image communication also homogenizes under within-modality baselines (CAF=0.834 [0.811, 0.858]), with comparable proportional effect; (3) group size moderates coupling direction -- K=5 produces homogenization while K=3 yields CAF > 1.0 (point estimates 1.14 and 1.06, CI pending), suggesting a directional shift toward diversification; (4) cross-model replication shows extreme variation (CAF 0.034-0.803), with DeepSeek dominated by format artifacts; (5) coupling is stateless -- driven by prompt context rather than cumulative updating, with continuous consensus producing monotonic convergence. These results establish LLM agent coupling as real, measurable, and controllable at the prompt level, with direct implications for multi-agent system design.

## 6. 论文的主要结论与发现

1. **文本通信导致显著同质化**（CAF=0.803，d=1.30，p<0.001），通过无通信消融（CAF=0.978，不显著）和提示扰动控制（CAF=1.082，反而多样化）证实**社交特异性**。
2. **图像通信在模态内基线下也呈现同质化**（CAF=0.834），与文本效应量比例相当，但绝对同质化幅度更大（因图像基线多样性更高）。
3. **组大小调节耦合方向**：K=5产生同质化（CAF<1），K=3产生CAF>1（点估计1.14和1.06），暗示存在临界组大小。

5. **耦合是无状态的**：基于提示上下文的即时驱动而非累积更新——交替共识产生锯齿形JSD轨迹，持续共识产生单调收敛至接近零的JSD。
6. **与经典意见动力学模型（DeGroot）定量偏离**：模拟预测更强同质化（CAF=0.657），且假设累积更新，与实验的无状态特性矛盾。

- **动态分析揭示机制**：通过交替/持续共识实验证明耦合的无状态性，挑战了传统意见动力学假设。

## 8. 不足与局限

- **自我锚定混淆**：协议消息中包含智能体自身先前的混合输出，可能独立于邻居影响而降低JSD（自一致性效应）。未进行“自身输出+权重，无邻居信息”的控制，导致CAF同质化估计可能高估真实社会耦合。
- **组大小与提示长度混淆**：大K增加提示中邻居摘要的token数，可能独立地影响锚定效应。未使用定长摘要隔离组大小。
- **模态与刺激存在部分混淆**：文本条件无显性刺激，图像条件有视觉输入。虽然模态内CAF校正了基线差异，但刺激有无本身可能影响结果（小规模试点表明无刺激设计更保守）。
- **多样性引入方式可能不自然**：使用不同人设和温度来防止基线多样性过低，但未进行统一温度消融实验（计划未来完成）。
- **无状态测量协议的协议差异**：动力学分析使用10类概率分布而非主实验的5类，数值不能直接比较；计划用5类协议复制。
- **单一任务领域**：仅使用性别刻板印象评估，耦合动态可能在推理、代码生成等任务中不同。
- **跨模型比较受格式伪影污染**：DeepSeek结果受约束JSON输出和禁用推理主导，不是真实社会耦合。自由格式协议的跨模型比较尚未进行。
- **CAF比值偏差**：在小样本（如N=10或N=5）时可能偏差更大，作者标注为初步结果。
- **CAF度量依赖性**：依赖于类别定义和数量，但嵌入空间JSD与这些测量高度相关（r>0.95），提供了鲁棒性证据。

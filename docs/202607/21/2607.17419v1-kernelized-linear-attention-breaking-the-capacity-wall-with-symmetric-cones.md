## 摘要
线性注意力承诺常数时间的循环推理，但在关联召回上性能急剧下降。我们将注意力召回形式化为球面填充问题，并引入核化线性注意力激活（KATA）框架，其特征映射基于第一性原理，通过自对偶齐次锥认证非负注意力权重。基于这一观察，我们证明秩一正半定（PSD）特征在容量与干扰之间提供了有利的权衡。KATA恢复了无参数凸输出门，并通过Welch干扰下限刻画关联容量。对于高于该下限的容忍度，KATA在不增加参数的情况下扩大状态，并允许在投影维度上具有指数多个键的球面编码。我们在两个操作点将KATA实现为融合Triton内核：一个flash-attention风格的前向，吞吐量高达FlashAttention-2的约1.6倍；以及一个精确的O(T)分块状态形式，在131k个token上达到FlashAttention-2前向吞吐量的约11倍。一阶特征的关联扫描将块间递归深度降低到O(log(T/C))（块大小C），平均吞吐量为匹配的序列线性注意力基线的约2.4倍。在长程MQAR和重复键覆盖上，多个KATA变体优于Gated DeltaNet，参数数量和状态大小与精度一同报告。归纳保持近乎完美的召回，而内核基准测试表明这些映射可以高效实现。KATA在16倍分布外长度下保持0.985的MQAR，接近使用约四分之一KV缓存条目的softmax。在340M参数LLM上的实验揭示了特征依赖的流畅性权衡，并阐明了位置嵌入、增量规则和衰减门如何与特征几何交互。

## Abstract
Linear attention promises constant-time recurrent inference but degrades sharply on associative recall. We formulate attention recall as a spherical-packing problem and introduce Kernelized Linear Attention Activations (KATA), a framework whose feature maps are derived from first principles by certifying nonnegative attention weights through a self-dual homogeneous cone. Building on this observation, we show that rank-one positive semi-definite (PSD) features offer a favorable capacity--interference tradeoff. KATA recovers a parameter-free convex output gate and characterizes associative capacity through the Welch interference floor. For tolerances above this floor, KATA enlarges the state without adding parameters and admits spherical codes with exponentially many keys in the projection dimension. We implement KATA as fused Triton kernels at two operating points: a flash-attention-style forward up to ${\sim}1.6\times$ FlashAttention-2 throughput, and an exact $O(T)$ chunked-state form that reaches ${\sim}11\times$ FlashAttention-2 forward throughput at $131$k tokens. An associative scan of the first-order feature lowers the inter-chunk recurrence depth to $O(\log(T/C))$ for chunk size $C$ and averages ${\sim}2.4\times$ the throughput of a matched sequential linear-attention baseline. On long-range MQAR and repeated-key overwrite, several KATA variants outperform Gated DeltaNet, with parameter counts and state sizes reported alongside accuracy. Induction preserves near-perfect recall, while kernel benchmarks show that the maps can be implemented efficiently. KATA retains $0.985$ MQAR at a $16\times$ out-of-distribution length, approaching the softmax with roughly one quarter of the KV-cache entries. Experiments on 340M-parameter LLMs reveal a feature-dependent fluency trade-off and clarify how positional embeddings, delta rules, and decay gates interact with feature geometry.

## 1. 核心问题与整体含义（研究动机和背景）
- **问题**：线性注意力（Linear Attention）虽具备理论上的常数时间递归推理复杂度，但在需要精确关联召回（associative recall）的任务上性能急剧下降，例如多查询关联召回（MQAR）和长程依赖场景。根本原因是固定大小的递归状态（recurrent state）面临**内存冲突**（memory collision）：大量键-值绑定必须共享同一个状态，造成检索退化。
- **背景**：现有线性时间替代方案（如 DeltaNet、Gated DeltaNet、Mamba2）通过引入遗忘门、未归一化写入和内容寻址擦除操作缩小了与 softmax 的差距，但其特征映射背后的几何机制依然不明确——什么限制了固定状态大小下的召回容量？哪些有限映射保留了 softmax 的各向同性？
- **本文目标**：从**球面填充问题**（spherical packing）视角统一理解容量瓶颈，利用**自对偶齐次锥**（symmetric cones）从第一性原理导出非负特征映射，提出 **KATA**（Kernelized Linear Attention Activations）框架，在理论上和硬件实现上突破线性注意力的容量瓶颈。

## 2. 方法论：核心思想、关键技术细节和算法流程
### 2.1 核心思想
- 将注意力召回模型化为**离散球面填充**：在单位球面上放置尽可能多的键特征向量，同时控制它们之间的最大干扰（mutual coherence / interference）。
- 通过**Koecher-Vinberg 分解**，**自对偶齐次锥**（正交锥、Lorentz 锥、正半定 PSD 锥）穷尽了实对称且容纳非负内积的几何，这些锥直接限制了可实现的容量上界。
- 提出**秩一 PSD 特征**（rank-one PSD feature）ψ(u) = uu⊤，特征维度为 p(p+1)/2（仅需 p 个投影坐标），其内积为 ⟨ui, uj⟩²，将原始线性内积平方，从而将干扰转化为原始向量的**绝对值小于√ϵ**的条件，允许在容忍度 ϵ 内实现指数级的键容量（定理4）。

### 2.2 关键技术细节
- **输出门自动导出**：归一化的线性注意力中，分母自然地构成一个**无参数凸输出门**（Proposition 1）：当前输出是历史归一化读出的凸组合与当前值的凸组合，无需额外参数。
- **容量理论**：
  - 正交锥（Orthant）：继承 Welch 下界，容量 O(d)。
  - Lorentz 锥：满足 Rankin 墙（Theorem 13），容量 O(d)（严格小于 2d）。
  - PSD 锥（秩一射线）：> Welch 下界后，通过球冠构造可得指数级容量（Theorem 4）。
  - 利用**Welch 干扰下界**和**功率信噪比（pSNR）** 分析比较线性特征、PSD 特征和 softmax 的理想化检索阈值（Theorem 7, Proposition 8）。
- **构造性填充**：提供基于互无偏基（MUB）和 DeVore 多项式的确定性有限字典，保证在最坏情况下的干扰上界。
- **硬件高效实现**：
  - 开发融合 Triton 内核，支持两种操作模式：
    - **Flash-KATA**：O(T²) 前向，吞吐量达 FlashAttention-2 的 1.6×。
    - **KATA-SSM**：O(T) 线性状态，使用分块精确 PSD 递归，在 131k token 时达到 11× FlashAttention-2 吞吐量。
  - 对于线性特征（ψ(x)=x），提出**关联扫描（associative scan）** 将块间递归从 O(N_C) 降低到 O(log N_C)，平均吞吐量为序列线性注意力基线的 2.4×。

### 2.3 算法流程（文字说明）
1. 将输入 Q、K 投影到单位球面，应用特征映射（如 PSD 的秩一外积）。
2. 维护递归状态 S_t = S_{t-1} + ψ(k_t)v_t⊤ 和 Z_t = Z_{t-1} + ψ(k_t)。
3. 读取时计算归一化输出 o_t = ψ(q_t)⊤ S_t / ψ(q_t)⊤ Z_t（实现凸门控）。
4. 在分块实现中，将序列分为块大小为 C 的 N_C 块；块内用密集矩阵乘法处理因果注意力，块间按照累加前缀和传播，或利用关联扫描加速。

## 6. 论文的主要结论与发现
1. **核化线性注意力（KATA）显著提升了固定状态下的关联召回容量**：在 MQAR 16× 外推中 KATA-M1 保留 0.985 准确率（接近 softmax 的 1.000），而状态大小仅为 softmax 所需 KV 缓存的约 1/4。

3. **Delta 规则和门控通过不同机制补充容量**：Delta 规则通过内容擦除解决重复写入歧义；Gated 门控通过位置加权提升最近值偏好。
4. **输出门是天然的结果，无需额外参数**：归一化线性注意力的分母自动产生凸插值门。
5. **硬件实现高效**：Triton 内核在广泛长度上达到与或优于 FlashAttention-2 的吞吐量，关联扫描进一步加速小批量场景。
6. **语言模型预训练表明 KATA 在大部分基准上与 softmax 可比，但在高熵 UUID 召回上显著优于 Gated DeltaNet**，不过在某些上下文理解任务（如 SQuAD）中受限于生成简洁性而得分较低。

## 7. 优点
- **理论奠基**：首次将线性注意力的非负性、容量限制与对称锥分类和球面填充理论系统化，提供了从第一性原理推导特征映射的通用框架。
- **统一性**：同时解释了正交锥、Lorentz 锥、PSD 锥的容量差异，并利用 Welch/Rankin 界给出精确容量定律，还扩展到温度 softmax。
- **硬件对齐设计**：特征映射和递归设计考虑了实际 GPU 的 SRAM 和 HBM 限制，融合内核实现生产级效率。
- **丰富的实验验证**：从合成任务到 340M 参数预训练，从精确字符串恢复到大模型基准，多层次评估，提供详细报告（含消融）。
- **开放代码**：发布 Triton 内核（GitHub），促进了可复现性。

## 8. 不足与局限
- **实锥限制**：仅考虑实对称锥（正交、Lorentz、PSD），未涵盖复 Hermitian、四元数 Hermitian 和例外因子（Albert 代数），可能遗漏更高容量结构。
- **PSD 特征的状态尺寸**：即使使用块分组（KATA-Mg），其状态仍比原始线性特征大一个数量级（如 266k vs 8k），限制了长序列下的内存可用性。
- **学习困难**：实际训练中，更高维的单一因子（KATA-M1）比多因子组合（KATA-M2）更难优化，导致某些任务上性能反而不如后者，理论上的容量优势未完全发挥。
- **语言模型上下文理解任务表现不足**：在 SQuAD、FDA 等需要语境推理和候选选择的场景，KATA 得分低于 softmax 和 Gated DeltaNet，部分归因于生成简洁性（长度惩罚）和模型困惑度较高。
- **未探索 GatedKATA 和 GatedDeltaKATA 的大规模预训练**：论文仅测试了门控在合成任务上的效果，尚未在 340M 语言模型上系统预训练门控变体。
- **外推长度受限**：依赖 RoPE 的 KATA 在外推到训练长度之外时性能下降明显，而 Gated DeltaNet 的乘法性遗忘门在外推上表现更好，但牺牲了精确召回。
- **实验平台限制**：仅基于 H100 和 A100，未涉及其他架构（如 AMD GPU、TPU）。

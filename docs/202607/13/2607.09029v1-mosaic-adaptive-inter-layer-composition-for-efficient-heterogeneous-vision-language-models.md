## 摘要
视觉-语言模型（VLM）通过使用同构Transformer处理多媒体数据取得了成功。近期研究表明，交织高效机制（如线性注意力）的异构结构在同构设计基础上同时提升了性能和推理延迟。然而，这些工作依赖于手工设计的静态混合模式，这种模式次优且难以适应特定硬件。为弥合这一差距，我们提出了面向自适应层间组合的多目标搜索（MOSAIC），这是一种硬件感知的搜索方法，可自动将同构模型转化为优化后的异构架构。MOSAIC将多种高效机制（包括线性、稀疏和低秩算子）集成到统一的搜索空间中。通过将选择问题建模为多目标混合整数规划（MIP）问题，我们的方法能够识别出在严格硬件延迟约束下最大化下游性能的最优配置。为缓解结构变换带来的性能下降，我们引入了一个两阶段参数恢复过程：首先进行全局离策略蒸馏以稳定内部表示，随后进行双教师在策略蒸馏，利用235B的oracle模型进行知识扩展，同时利用原始的4B教师模型保持分布稳定性。我们通过从Qwen3-VL-4B-Instruct衍生的MOSAIC-4B验证了MOSAIC。结果表明，MOSAIC-4B在多个基准测试中匹配了基线的性能，同时仅需原始训练成本的不到2%。此外，它显著提升了推理效率，实现了1.76倍预填充和2.54倍解码加速。

## Abstract
Vision-Language Models (VLMs) have achieved success using homogeneous Transformers to process multimedia data. Recent studies show that heterogeneous structures interleaving efficient mechanisms, like linear attention, improve both performance and inference latency over homogeneous designs. However, these efforts rely on handcrafted static mixing patterns, which are sub-optimal and difficult to adapt to specific hardware. To bridge this gap, we propose Multi-Objective Search for Adaptive Inter-layer Composition (MOSAIC), a hardware-aware search method that automatically transforms homogeneous models into optimized heterogeneous architectures. MOSAIC integrates diverse efficiency mechanisms--including linear, sparse, and low-rank operators--into a unified search space. By formulating the selection as a multi-objective Mixed Integer Programming (MIP) problem, our method identifies optimal configurations that maximize downstream performance under strict hardware latency constraints. To mitigate performance degradation from structural transitions, we introduce a two-stage parameter recovery process: global off-policy distillation to stabilize internal representations, followed by a dual-teacher on-policy distillation leveraging a 235B oracle for knowledge expansion and the original 4B teacher for distributional stability. We validate MOSAIC through MOSAIC-4B, derived from Qwen3-VL-4B-Instruct. Results demonstrate that MOSAIC-4B matches the baseline's performance across multiple benchmarks while requiring less than 2% of the original training cost. Furthermore, it substantially improves inference efficiency, achieving 1.76x prefilling and 2.54x decoding speedups.

## 1. 核心问题与整体含义

- **研究动机**：现有视觉-语言模型（VLM）普遍采用同构Transformer架构（如密集自注意力+MLP），其二次时间复杂度在长上下文或资源受限硬件上导致高昂计算开销。近期异构架构（如交织线性注意力、稀疏注意力的设计）虽然展现了性能与效率优势，但**依赖手工设计的静态混合模式**（如每4层插入1个替代层），这种启发式方案存在两个关键瓶颈：
  - 手动排列无法探索巨大架构空间，难以达到特定任务的最优性能-效率帕累托前沿；
  - 固定配置无法灵活适配不同硬件平台（不同硬件对不同算子延迟特性不同），导致要么过慢要么浪费算力。
- **整体意义**：本文提出**MOSAIC**，将同构VLM自动转化为针对硬件延迟约束优化的异构架构，首次实现面向硬件的自适应异构组合搜索，兼顾下游性能与推理效率。

## 2. 方法论

### 2.1 核心思想

通过**三阶段流程**实现自动化异构设计：
1. **统一搜索空间构建**：每个Transformer层包含注意力子块和FFN子块的可选算子集合。
2. **多层多目标架构搜索**：利用块级局部蒸馏初始化候选块，基于多能力评分（PPL、KL散度、LLM基准、VLM基准）和硬件运行时建模，将选择问题形式化为**多目标混合整数规划（MIP）**，在给定延迟预算下寻找最优组合。
3. **两阶段参数恢复**：先全局离策略蒸馏稳定内部表示，再结合大容量oracle教师（235B）和原教师（4B）进行双教师在策略蒸馏，弥补结构变化带来的性能损失。

### 2.2 关键技术细节

- **搜索空间**：
  - 注意力子块：Grouped-Query Attention (GQA)、Kimi Delta Attention (KDA)、Gated Delta Network (GDN)、Multi-head Latent Attention (MLA)、Sliding Window Attention (SWA)。
  - FFN子块：对SwiGLU中间维度进行结构化剪枝，缩放比例从0%到100%共7档（0%表示移除FFN）。
  - 共35种每层配置，教师模型Qwen3-VL-4B-Instruct有36层，总搜索空间35^36。

- **块级局部蒸馏（BLD）**：每个候选块独立对齐该层教师的输出激活，使用归一化MSE损失。输入来自前一层教师输出，可高度并行。仅需80K训练样本。

- **多能力评分**：对每个候选块，计算相对能力分数S_{k,l,i} = (-1)^{σ_k} * (P_{k,l,i} - M_k) / M_k，其中M_k为参考值（教师性能或最小KL值），σ_k指示指标方向（KL和PPL为低更好，基准准确率为高更好）。能力集Λ包含PPL、KL、LLM基准（MMLU）、VLM基准（SeedBench）。

- **多目标MIP求解**：目标函数为切比雪夫标量化后的最大最小问题，引入辅助变量C，转化为线性MIP，使用python-mip求解器。延迟约束来自每个候选块的实测运行时间（实验设1.5×加速比）。

- **两阶段参数恢复**：
  - **阶段1：全局离策略蒸馏**。先在预训练数据（10M文本+10M图像-文本对）上使用对称KL散度+交叉熵损失蒸馏；再在指令数据（~10M）上继续蒸馏。
  - **阶段2：双教师在策略蒸馏**。对于每个提示，学生自回归生成响应，235B oracle教师提供SFT目标（交叉熵损失），4B原教师计算学生生成token上的logits（对称KL损失），两者加权组合。数据量约2M。

### 2.3 公式与算法流程

- 对齐损失：L_align = ||o_t - o_c||_2 / ||o_t||_2
- 能力分数：S_{k,l,i} = (-1)^{σ_k}·(P_{k,l,i} - M_k)/M_k
- MIP：min_C max_k [w_k·(z_k - F_k(X))] 且满足延迟约束。
- 全局蒸馏损失：L_recover = L_KL(p||q_θ)+L_KL(q_θ||p) + α·L_CE
- 双教师损失：L_align = L_KL(θ; D') + β·L_CE(θ; D_o)

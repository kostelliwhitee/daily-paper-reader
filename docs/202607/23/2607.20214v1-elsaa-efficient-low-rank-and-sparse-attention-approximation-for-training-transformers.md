## 摘要
二次方$N\times N$的注意力得分矩阵仍然是扩展Transformer到更长输入长度的主要障碍。现有的高效注意力方法通常通过施加稀疏性（使每个查询只关注一小部分键）或使用低秩/核技巧（将全局交互压缩到低维表示）来缓解这一瓶颈。我们提出了\emph{ELSAA}，一种高效的注意力低秩与稀疏近似方法。重要的是，ELSAA并\emph{不}将Transformer中学习的投影矩阵或输出矩阵分解为稀疏和低秩因子。相反，在密集投影生成$Q,K,V$后，ELSAA近似了诱导出的注意力得分算子本身：一个稀疏分支捕获选定的高相似度交互，而一个低秩分支总结分散的全局交互。由于这两个分支可以在具有非常不同分母质量的支撑集上进行归一化，ELSAA引入了一个分母感知的融合项，根据其相对于低秩分支估计的注意力质量来缩放稀疏分支。这提供了一个实用的框架来构建低秩和稀疏注意力输出，而无需生成完整的二次得分矩阵，旨在实现更长上下文的训练，同时保留尖锐的token级交互和广泛的上下文混合。

## Abstract
The quadratic $N\times N$ attention score matrix remains a central obstacle to extending Transformers to longer input lengths. Existing efficient attention methods usually reduce this bottleneck by either imposing sparsity, so that each query attends to only a small subset of keys, or by using low-rank/kernel sketches, so that global interactions are compressed into a lower-dimensional representation. We propose \emph{ELSAA}, an efficient low-rank and sparse approximation of attention. Importantly, ELSAA does \emph{not} decompose the learned projection or output matrices of the Transformer into sparse and low-rank factors. Instead, after dense projections produce $Q,K,V$, ELSAA approximates the induced attention score operator itself: a sparse branch captures selected high-similarity interactions, while a low-rank branch summarizes diffuse global interactions. Since the two branches can be normalized over supports with very different denominator mass, ELSAA introduces a denominator-aware fusion term that scales the sparse branch according to its estimated attention mass relative to the low-rank branch. This gives a practical framework for constructing low-rank and sparse attention outputs without materializing the full quadratic score matrix, aiming to enable longer-context training while preserving both sharp token-level interactions and broad contextual mixing.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：Transformer 中自注意力机制的 $N \times N$ 得分矩阵具有二次复杂度，成为处理长输入序列的主要瓶颈。
- **现有方案局限**：
  - **稀疏注意力**（如局部窗口、哈希选择）能捕获高相似性交互，但可能丢失扩散的全局信息。
  - **低秩/核近似**（如 Performer、RACE）能压缩全局交互，但可能平滑掉尖锐的 token 级依赖。
  - 简单相加两类分支会导致分母质量不同的尺度失配，影响模型训练稳定性。
- **研究目标**：提出一种同时结合稀疏与低秩优势的注意力近似方法，在不生成完整二次矩阵的前提下，兼顾局部高相似性交互与全局上下文混合，从而支持更长上下文的训练。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：将注意力得分算子分解为**稀疏分支**（精确计算高相似度查询-键交互）和**低秩分支**（用压缩特征总结全局扩散交互），并通过**分母感知融合**动态平衡两者贡献。
- **关键技术细节**：
  1. **稀疏分支**：使用排序局部敏感哈希（sortLSH）将查询和键按哈希值排序，分组为等大小块，在每个块内执行精确注意力计算。支持因果变体使用递归分治结构。
  2. **低秩分支**：采用 RACE 注意力，通过软哈希桶将键/值聚合到桶特征中，查询从桶摘要中读取，实现线性复杂度全局上下文。
  3. **分母感知融合**：对于每个查询 $i$，引入稀疏乘子 $m_{\text{sparse},i} = d_{\text{sparse},i} / (d_{\text{sparse},i} + \lambda_i d_{\text{lr},i} + \epsilon)$，其中 $d_{\text{sparse}}, d_{\text{lr}}$ 分别为两分支的分母代理，$\lambda_i$ 可固定或学习。最终输出为 $O_i = g_{\text{sparse},i} m_{\text{sparse},i} O_{\text{sparse},i} + g_{\text{lr},i} O_{\text{lr},i}$，$g$ 为可学习门控。
  4. **分析**：论文通过秩分析证明，稀疏+低秩混合算子可以达到满秩的概率很高，只要稀疏碰撞图的最大匹配大小不小于 $n-r$，低秩部分填补剩余维度。
  5. **复杂度**：总交互量 $O(N(s + L_s 2^\gamma))$，相对精确注意力的 $O(N^2)$ 近似线性，例如 $N=32000$ 时可减少约 99% 的交互。

## 6. 论文的主要结论与发现

- **分母感知融合有效**：ELSAA 在长上下文结构化视觉任务上显著优于无融合混合 baseline（+1.3% 平均测试准确率），尤其在 Oxford-IIIT Pet、Flowers-102 上提升明显。
- **任务特性决定分支重要性**：视觉注意力呈尖锐局部分布，稀疏分支至关重要；长文本主题分类呈扩散弱证据积累，低秩分支更优。ELSAA 能够处理两种场景。
- **极长上下文下精确注意力失败**：在 64K 检索任务中，ExactFlash 训练/测试准确率均约 50%（随机），而 ELSAA 和 RACE 达到 65% 以上，说明近似注意力可作为结构性先验稳定优化。
- **长度外推能力强**：在 NIAH 任务中，ELSAA 在训练长度（1024）以外的所有长度（512~65536）均保持高准确率，而 RACE 在 4K 以上骤降，Sparse LSH 在短于训练长度时失败。
- **因果场景匹配或超越基线**：因果 ELSAA 在 ArXiv 64K 上优于因果精确注意力和 RACE，在 Tiny ImageNet 上接近 ExactFlash。
- **计算效率高**：总交互量近似线性于序列长度，参数开销可忽略不计。
